# ESP32 AI CAM

AI Thinker ESP32-CAM live video server.

Features:
- MJPEG live stream
- Photo capture
- Flash LED control
- Arduino IDE support

## Setup

1. Install ESP32 board support in Arduino IDE.
2. Select `AI Thinker ESP32-CAM`.
3. Edit WiFi settings in the sketch.
4. Upload and open Serial Monitor.
5. Visit the printed IP address.

Stream URL:
`http://ESP32-IP:81/stream`

GitHub Pages can host the control page, but the video stream comes from the ESP32.
